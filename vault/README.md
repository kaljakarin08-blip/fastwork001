# Vault — Knowledge Base

นี่คือ **คลังความรู้** ที่ Hermes ใช้สร้าง content

## วิธีเพิ่มข้อมูล

1. วางไฟล์ `.txt` หรือ `.md` ลงในโฟลเดอร์ที่เหมาะสม
2. รีสตาร์ทแอป หรือกด **"Re-index Vault"** บน Dashboard → Vault

```
vault/
├── law/          ← ข้อมูลกฎหมาย (กฎหมายแพ่ง, อาญา, แรงงาน ฯลฯ)
├── accounting/   ← ข้อมูลบัญชี/ภาษี (ภงด, VAT, มาตรฐานบัญชี ฯลฯ)
└── general/      ← ข้อมูลทั่วไปของสำนักงาน (profile, FAQ)
```

## รูปแบบไฟล์ที่รองรับ

| ชนิด | รองรับ | หมายเหตุ |
|------|--------|---------|
| `.md` | ✅ | แนะนำ — อ่านง่าย จัดหัวข้อได้ |
| `.txt` | ✅ | ข้อความธรรมดา |
| `.pdf` | 🔜 | เร็วๆ นี้ |

## ขนาดไฟล์แนะนำ

- **1 ไฟล์ = 1 หัวข้อ** (ไม่รวมหลายเรื่องในไฟล์เดียว)
- ขนาดไม่เกิน 50,000 ตัวอักษร/ไฟล์
- ระบบจะตัด chunk อัตโนมัติทุก ~500 tokens
